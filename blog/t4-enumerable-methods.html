The Value of Cycle

A Ruby Enumerable Method

8/3/15

Today I learned about a Ruby enumerable method today called cycle. A short explanation of cycle: when given a collection of items, cycle passes each item to a block of code an infinite number of times until told to break. Cycle is a convenient way to create loops and break them if a specific condition is met. Cycle can also accept an argument as a positive integar which will loop through a collection as many times as you command. Here is an example:

dogs = ["Buddy", "Fido", "Murphy"]

dogs.cycle { |x| puts x }

Buddy

Fido

Murphy

Buddy

Fido

Murphy

... forever

Using the same array dogs, here is an example when you give cycle an argument:

dogs.cycle(3) { |x| puts x }

Buddy

Fido

Murphy

Buddy

Fido

Murphy

Buddy

Fido

Murphy

This time cycle only ran through the array three times. This should give you a basic understanding of the cycle method. Next time you need to create a loop, give cycle a try!

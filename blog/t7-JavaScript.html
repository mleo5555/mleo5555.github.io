JavaScript vs. Ruby

8/23/15

Having experience with Ruby made it much easier to learn JS. A key element that Ruby and JavaScript share is they are both object oriented languages. JS and Ruby share many similarities but some subtle differences in sytax can really throw you off. Ruby has more built in methods to work with and reads a little more like english in my opinion. Iterating over objects and looping are slightly different in both languages as well. One thing that stood out the most to me about JS was the absence of hashes.

Ruby Hashes vs. JavaScript Objects

In JS, objects function very similarly to how hashes function in Ruby.
